# Greenhouse Gas Emission Prediction

## Overview

This project uses Machine Learning techniques to predict greenhouse gas emissions using environmental and industrial datasets. The system helps identify emission trends and supports sustainability-focused decision-making.

## Features

- Greenhouse Gas Emission Prediction
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modeling
- Streamlit Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Streamlit

## Models Used

- Linear Regression
- Random Forest Regression

## Results

- Random Forest achieved better prediction performance than Linear Regression.
- Identified key factors influencing greenhouse gas emissions.
- Generated sustainability-focused insights using predictive analytics.
